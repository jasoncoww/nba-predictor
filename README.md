# 🏀 NBA Game Outcome Predictor

AI-powered NBA game predictor built with Next.js. Uses win%, rest days, recent form, and live injury reports to predict game outcomes.
---

## How it works

- **`pages/index.jsx`** — The full predictor UI (team selectors, rest calculation, injury fetch, prediction display)
- **`pages/api/anthropic.js`** — Server-side proxy that adds your API key to Anthropic requests. The key is never sent to the browser.
- Rest days are calculated instantly from a hardcoded full-season schedule (Oct 2025 – Apr 2026)
- Injury reports are fetched live via Claude's web search tool
- Predictions are generated by `claude-sonnet-4-20250514` with win%, form, rest, and injury context

---

## Sharing

Once deployed, just send people your Vercel URL. No login required — they open it and start predicting.

If you want to limit usage (so your API bill doesn't get out of hand), add a simple password check in `pages/api/anthropic.js` or use Vercel's built-in password protection under Project Settings → Security.
